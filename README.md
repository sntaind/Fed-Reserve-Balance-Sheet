## 📥 데이터 입력 방법

### 방법 1: PDF 파일 업로드
1. [연준 H.4.1 페이지](https://www.federalreserve.gov/releases/h41/)에서 PDF 다운로드
2. 대시보드에서 **PDF** 버튼 클릭 또는 파일을 드래그 앤 드롭
3. 여러 파일을 한번에 업로드하여 시계열 데이터 구축 가능

### 방법 2: 텍스트 붙여넣기 (모바일 권장)
1. PDF를 다른 앱(Adobe Reader, 브라우저 등)에서 열기
2. 전체 선택 → 복사
3. 대시보드에서 **붙여넣기** 버튼 클릭
4. 텍스트 붙여넣기 후 **분석하기** 클릭

## 📊 탭 설명

| 탭 | 내용 |
|---|---|
| **개요** | 주요 지표 카드 (총자산, 지준금, TGA, RRP 등) |
| **준비금 요인** | Table 1 전체 항목 - 준비금 공급/흡수 요인 |
| **기록 항목** | Table 1A Memorandum Items |
| **만기 분포** | Table 2 증권 만기별 분포 |
| **대출/증권** | Table 4/4A 대출 및 증권대출 상세 |
| **재무제표** | Table 5 연결재무상태표 |
| **지역 연준** | Table 6 12개 지역 연준 개별 현황 |
| **연방준비권** | 연방준비권 발행 및 담보 현황 |
| **추이** | 시계열 차트 (2개 이상 데이터 필요) |

## ⚙️ 설정

- **테마**: 다크/라이트 모드 전환
- **글자 크기**: 작게/보통/크게/아주 크게
- **서체**: 본문 및 숫자 서체 선택

## 💾 데이터 저장

- 업로드한 데이터는 브라우저 로컬에 자동 저장됩니다
- **저장된 데이터** 버튼으로 이전 데이터 조회/삭제 가능
- 화살표 버튼(↑↓)으로 날짜별 데이터 탐색

## 📱 모바일 사용 팁

- iOS/Android에서 파일 선택이 안 될 경우 **텍스트 붙여넣기** 기능 사용
- PDF 뷰어 앱에서 전체 텍스트 복사 → 대시보드에 붙여넣기
